---
name: omikuji
description: おみくじを引いて、今日の運勢（大吉、中吉、小吉、吉、末吉、凶、大凶）を出力します。
---

## 実行ステップ

1. **おみくじを引く**
    - Python スクリプトを使用して、ランダムに運勢を決定します。

2. **結果を表示する**
    - 決定した運勢をユーザーに報告します。

## ツール・コマンド例

```powershell
# おみくじを引くコマンド
python -c "import random; print(random.choice(['大吉', '中吉', '小吉', '吉', '末吉', '凶', '大凶']))"
```
