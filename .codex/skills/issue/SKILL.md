# Issue 作成規約

- **出力言語は必ず日本語とすること。**
- ユーザーから「issue書いて」「チケットを書いて」等の Issue 作成の依頼があった場合は、以下の内容を
  **Markdown のコードブロック（生md）** で生成すること。

## 生成フォーマット

```markdown
**タイトル:** [タイトル]

**本文:**
## 概要
[概要]

## 主な対応内容
- [対応内容]

## 期待される効果
- [効果]
```
